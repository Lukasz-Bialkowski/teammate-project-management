Teammate Technology Stack

Technology stack includes:

  • Bootstrap
  • jQuery
  • Font Awesome
  • AngularJS
  • HTML 5
  • CSS 3
  • Java
  • Spring Framework

Teammate application was written in modern technologies. Our developers focused on creating secure application, which will work fast and be secured.

Service List

Project definition

Teammate allows managers to create project and describe them.

Mail communication

In Teammate lots of state changes is automaticaly send to others to ensure that recipient will be notified about project events, new documents, changes on taskboard etc.

User profiles

In Teammate everybody has unique profile which allow you to access application.

Task worklog

As a task contractor you should log your working hours. This information is crucial for your manager.

Taskboard

All tasks are displayed to you in taskboard. You can change task states by just drag&drop features.

Task definition

If you are a project member you can define tasks which is pinned to specific project. This task will be noticable in project taskboard.

Project documents

All project has some unique information, project documentation, newhire introduction. As a project member you are allowed to create document and public them to others.

Project events

In Teammate project members can create project events and inform others about it.

Secure information

Teammate is secured application and all personal details and account credentials are protected.
